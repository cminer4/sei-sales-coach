- **Required field behavior** — Agent Type in Prompt Control is required: save is blocked and a clear validation message is shown when no type is selected. Matches FR-002.
- **“Assigned to all” behavior** — When filtering by a specific Agent Type, documents assigned to all agents are always included (OR with docs for that type). Matches the clarified spec.

- **Prompt Control**: No extra requests; agent_type is loaded and saved with the existing agent payload. ✅
- **Scalability**: Agent type filter uses a small set of agent IDs; document query uses standard Prisma filters. Fine for current and near-term scale. If the number of agents grows very large, the “agents of type” query remains cheap (indexed by agentType if needed later).

No performance concerns for this feature.
